# Interactive Excel Sales Dashboard

## Overview

This project is an interactive sales dashboard created in Microsoft Excel. It helps analyze sales performance using Pivot Tables, Pivot Charts, Slicers, and VBA. Users can filter the data by city and view different sales metrics through four dashboard sections.

## How It Works

The dashboard uses Pivot Tables to summarize the data and Pivot Charts to visualize it. Users can select a city from the top navigation to update the dashboard. VBA is used to control the dashboard views and improve interaction between the different sections.
